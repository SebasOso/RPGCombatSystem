# RPGCombatSystem
 
# Animation Class:

The main scene is "Animacion".

## Inputs:
### Movement: 
- WASD Keyboard.
- Left Joystick Gamepad.
### Run:
- Shift Keyboard.
- L1 Gamepad.
### Attack:
- Left Clik Keyboard.
- R1 Gamepad.
### Change Character:
- "M" for "Maria" character in Keyboard.
- "J" for "John" character in Keyboard.
- Down Arrow for "John" character in Gamepad.
- Up Arrow for "Maria" character in Gamepad.
