## Inputs:
### Movement: 
- WASD Keyboard.
- Left Joystick Gamepad.
### Run:
- Shift Keyboard.
- L1 Gamepad.
### Aim:
- Right Clik Keyboard.
- L2 Gamepad.
### Move Aim:
- Arrows Keyboard.
- Arrows Gamepad.
### Jump:
- Space Keyboard.
- X Gamepad.
